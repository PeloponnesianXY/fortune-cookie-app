# Fortune paper prompt template

Subject: clean paper strip for dynamic text overlay or compositing

Prompt:

```text
single fortune paper strip, centered, slight natural bend, minimal blue end tabs, clean printable surface, soft stylized shading, same lighting direction as cookie assets, isolated for mobile UI compositing, transparent or plain removable background, designed to accept overlaid text
```

Negative prompt:

```text
printed fortune text, large lucky numbers, tabletop scene, hands, harsh shadows, curled into unreadable shape, multiple strips, wrinkles that block text, watermark, background clutter
```
